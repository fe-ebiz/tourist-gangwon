YEONGJIN BEACH

자가 차량 이용시

- 네비게이션 주소 또는 영진 해변 입력
- 강원도 강릉시 연곡면 해안로 주소를 입력

대중교통 이용시

- 강릉시외버스터미널 하차시
: 강릉시외고속버스 정류장에서 314번 버스 승차,
삼천리A 정류장에서 하차, 700m 이동
- 강릉역 하차시
: 강릉역 건너편 정류장에서 314번 버스 승차,
삼천리A 정류장에서 하차, 700m 이동
